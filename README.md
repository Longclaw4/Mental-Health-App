# MindTrack - Mental Health Tracking App

A mobile application built with React Native that helps users track their mental health through mood tracking, daily journaling, and analytics.

## Features

- Mood Tracking
- Daily Journal
- Insight Analytics
- User Authentication

## Tech Stack

- React Native
- TypeScript
- React Navigation
- Vector Icons

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

3. Run the app:
- For Android:
```bash
npm run android
```
- For iOS:
```bash
npm run ios
```
