# Mental Health App

A comprehensive mental health tracking application with both web and mobile interfaces.

## Features

- Mood Tracking
- Daily Journal
- Insight Analytics
- User Authentication

## Tech Stack

- React Native (Mobile)
- TypeScript
- React Navigation
- Vector Icons

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
